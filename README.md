## 📋 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 수행 기간 | 2024.10.01 ~ 2024.12.17 |
| 담당 역할 | PM/MM 모듈 설계 및 개발 |
| 개발 방식 | Module Pool, Report 프로그램 |

---

## 🔧 개발 프로그램 목록

### 📦 Module Pool
| 프로그램명 | 설명 |
|-----------|------|
| 원자재 품질검수 프로그램 | 입고 원자재 품질 기준 검수 화면 개발 |
| 구매오더 생성 프로그램 | PO 생성 및 관리 화면 개발 |
| 원자재 등록 프로그램 | 자재 마스터 등록/변경 화면 개발 |

### 📊 Report
| 프로그램명 | 설명 |
|-----------|------|
| 거래처 송장 업로드 프로그램 | 외부 송장 데이터 SAP 업로드 처리 |
| 송장 검증 프로그램 | 업로드 데이터 유효성 검증 및 오류 리포트 |
| 자재조회 프로그램 | 자재 현황 통합 조회 리포트 |
